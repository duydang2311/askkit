# Askkit

A unified, local desktop AI chatbot that lives in your system tray, ready to launch instantly whenever you need assistance.

## Tech Stacks

**Frontend:**

- Svelte & SvelteKit
- TypeScript
- Tailwind CSS
- Vite

**Backend:**

- Rust
- Sqlite

**Desktop Integration:**

- Tauri: currently Windows only.
